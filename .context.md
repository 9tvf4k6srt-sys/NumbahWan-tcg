## Lessons Learned (Don't Repeat!)

### L1: Missing Function Call
**Problem:** `switchSeason()` called `buildPool()` which didn't exist
**Fix:** Always grep for function before calling it
```bash
grep -n "function funcName" file.html
```

### L2: Touch Events Blocked
**Problem:** Carousel captured all touch events, blocking UI buttons
**Fix:** Check target in onDown():
```javascript
if (target.closest('.season-selector')) return;
```

### L3: Cards Array Not Updated
**Problem:** Loaded new DATA but didn't update `cards` array
**Fix:** After loadCardsData(), always:
```javascript
cards = [...DATA];
render();
```

### L4: S2 Images Don't Exist
**Problem:** cards-s2.json references "s2/xxx.webp" but folder empty
**Fix:** getThumbUrl() returns S1 placeholder based on rarity
