How It Works: Three Roles, One Pipeline

commit
→
LEARN
→
EVAL
→
FIX
→
VERIFY
→
done

Learner (gitwise + NW-Memory)

Silently observes every commit. Tracks which files break, which change together, what areas have constraints. Zero config — just commit.

Evaluator (built into Fixer)

Scores both systems on 7 dimensions each. Identifies weaknesses: repeat breakages, uncovered areas, shallow lessons, rising fix rates.

Fixer (nw-fixer.cjs)

Reads both brains, applies 8 fix actions, then re-evaluates to verify. Loops up to 3 times. Reports before/after scores.

Setup: 30 Seconds, Zero Dependencies

1

Drop three files into your repo

gitwise.cjs + nw-memory.cjs + nw-fixer.cjs

2

Install git hooks

node gitwise.cjs --install and node nw-memory.cjs --init

3

Just commit. The system learns from you.

Every commit triggers: learn → eval → fix → verify. No manual intervention ever needed.
