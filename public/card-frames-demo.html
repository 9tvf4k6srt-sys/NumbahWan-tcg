🎴 Card Frame System

Pure CSS frames - Zero image generation credits needed

💡 How It Works

  • Card artwork is full-bleed (edge-to-edge, no borders in image)
  • CSS applies unique animated frames per rarity tier
  • Name, stats, and rarity badge rendered via CSS overlay
  • One card image + CSS = Complete card display
  • Works on all pages automatically

Usage Code

/* HTML Structure */
<div class="nw-card" data-rarity="mythic">
  <img class="nw-card-art" src="/static/cards/your-card.jpg">
  <div class="nw-card-info">
    <div class="nw-card-name">Card Name</div>
    <div class="nw-card-stats">ATK 10 | DEF 8 | DIVINE</div>
  </div>
  <div class="nw-card-rarity">MYTHIC</div>
</div>

/* JavaScript (using renderer) */
