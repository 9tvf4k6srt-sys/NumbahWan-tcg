🎨 Pure CSS Frames

All frame designs use pure CSS - gradients, borders, shadows, and animations. No image generation needed per card.

⚡ Auto Stats Generation

Cards without explicit stats automatically receive balanced ATK/DEF values based on their rarity tier.

📱 Responsive Sizes

Three size variants (sm, md, lg) that adapt to any layout. All frames scale proportionally.

🔮 Rarity Effects

Each rarity has unique animated effects - divine flames for Mythic, gold shimmer for Legendary, arcane pulses for Epic, and more.

◆ Stat Gems

ATK/DEF displayed as gem badges with rarity-themed styling. Cost gem in top-right corner for mana/resource display.

🚀 Batch Rendering

Efficiently render hundreds of cards using DocumentFragment. Lazy loading for images to optimize performance.

💻 Usage Examples

// 1. Include the required files
<link rel="stylesheet" href="/static/nw-card-frames.css">
<script src="/static/nw-card-renderer.js"></script>

// 2. Render a single card
const card = {
    id: 1,
    name: 'RegginA, The Eternal Flame',
    rarity: 'mythic',
    img: '01-reggina-mythic.jpg',
    stats: 'ATK 10 | DEF 8'
};
const cardEl = NW_CARDS.render(card);
document.body.appendChild(cardEl);

// 3. Batch render multiple cards
NW_CARDS.renderCards(cardsArray, '#container', { size: 'md' });

// 4. Generate HTML string (for SSR)
const html = NW_CARDS.generateHTML(card);

// 5. Auto-render from data attributes
<div data-nw-cards='[...]' data-nw-size="sm"></div>
