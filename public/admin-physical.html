Code Format

                
Codes follow the format: OG{PRINT#}{RARITY}{SALT}{CHECKSUM}

                
    
                    
  • OG - Set prefix (Origins)
    • 
                    
  • PRINT# - 4-digit print number (0001-9999)
    • 
                    
  • RARITY - 2-char rarity code (MY, LG, EP, RA, UC, CO)
    • 
                    
  • SALT - 4-char random salt
    • 
                    
  • CHECKSUM - 4-char CRC16 validation
    • 
                

                
Example: OG0047MYA7B23F4D
