## Card Rendering

### Single Card
```html
<div id="card-container"></div>
<script>
const card = {
  id: 1,
  name: 'Card Name',
  rarity: 'mythic', // mythic|legendary|epic|rare|uncommon|common
  img: 'filename.jpg',
  stats: 'ATK 10 | DEF 8 | COST 7'
};
const el = NW_CARDS.render(card, { size: 'md' }); // sm|md|lg
document.getElementById('card-container').appendChild(el);
</script>
```

### Card Grid (Multiple Cards)
```html
<div class="card-grid" id="myGrid"></div>
<script>
const cards = [...]; // Array of card objects
NW_CARDS.renderCards(cards, '#myGrid', { 
  size: 'md',
  onClick: (card, el) => console.log('Clicked:', card.name)
});
</script>
```

### Cards by Rarity
```html
<div id="mythicCards"></div>
<script>
fetch('/static/data/cards.json')
  .then(r => r.json())
  .then(data => {
    const mythics = data.cards.filter(c => c.rarity === 'mythic');
    NW_CARDS.renderCards(mythics, '#mythicCards', { size: 'md' });
  });
</script>
```

### Card with Click Handler
```javascript
const el = NW_CARDS.render(card, {
  size: 'md',
  onClick: (card, element) => {
    // Show modal, navigate, etc.
    console.log('Selected:', card.id, card.name);
    element.classList.add('selected');
  }
});
```

## Wallet Integration

### Get Guest ID
```javascript
const guestId = NW_WALLET.getGuestId();
console.log('Guest:', guestId); // e.g., "guest_abc123"
```

### Save/Load Forge State
```javascript
// Save
NW_WALLET.saveForgeState({
  pulls: 150,
  pity: { mythic: 45, legendary: 12 },
  lastPull: Date.now()
});

// Load
const state = NW_WALLET.getForgeState();
```

### Collection Management
```javascript
// Add card to collection
NW_WALLET.addToCollection(cardId);

// Get collection
const collection = NW_WALLET.getCollection(); // Array of card IDs

// Check if owned
const owned = NW_WALLET.hasCard(cardId); // boolean
```

## API Calls

### Fetch All Cards
```javascript
fetch('/static/data/cards.json')
  .then(r => r.json())
  .then(data => {
    console.log('Total:', data.totalCards);
    console.log('Cards:', data.cards);
  });
```

### Pull Cards (Gacha)
```javascript
fetch('/api/cards/pull', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify({ count: 1 }) // 1, 5, or 10
})
.then(r => r.json())
.then(result => {
  console.log('Pulled:', result.cards);
});
```

### Get Cards by Rarity
```javascript
fetch('/api/cards/rarity/mythic')
  .then(r => r.json())
  .then(cards => console.log('Mythics:', cards));
```

## Common Fixes

### Card Image Not Showing
1. Check z-index: `.nw-card-inner` should be `z-index: 2`
2. Check image path: Should be `/static/cards/filename.jpg`
3. Check file exists: `ls public/static/cards/filename.jpg`

### Page 404
1. Add to `staticPages` array in `src/index.tsx:1314`
2. Run `npm run build && pm2 restart all`

### CSS Changes Not Applying
1. Hard refresh: Ctrl+Shift+R
2. Check dist is rebuilt: `npm run build`
3. Check file is in dist: `ls dist/static/`
