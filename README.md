# NumbahWan Guild Website

> "We are not just a guild, but FAMILY"

## Project Overview
- **Name**: NumbahWan Guild
- **Game**: MapleStory Idle RPG (TW Server)
- **Goal**: Stunning guild website showcasing our journey to #1
- **Features**: Member roster, CP race leaderboard, progress tracker, guild fun gallery

## Live URLs

- **Production**: (Deploy to Cloudflare Pages)

## Features

### Hero Section
- Animated pixel art "N" emblem (exact replica of guild emblem)
- GSAP-powered text animations with neon glow effects
- Floating particle system
- Aurora gradient background

### Member Roster ("The Family")
- 12 guild members with full stats
- Level, CP, Contribution, Upgrade display
- Role badges (Master, Vice Master, Leader, Member)
- Online status indicators
- Hover animations on cards

### CP Race Leaderboard
- Visual bar comparison of all members
- Animated progress bars on scroll
- Medal rankings (Gold, Silver, Bronze)
- Real-time style updates

### Road to #1 Progress
- Guild Level progress
- Total Guild CP tracker
- Member capacity
- Boss raids completed
- Server ranking progress
- Milestone achievements

### Guild Shenanigans (Photo Gallery)
- Placeholder for AI-generated photos
- Glassmorphic photo cards
- Upload photo feature (planned)

## Tech Stack
- **Framework**: Hono.js
- **Platform**: Cloudflare Pages
- **Styling**: TailwindCSS (CDN)
- **Animations**: GSAP + ScrollTrigger
- **Fonts**: Press Start 2P (pixel), Orbitron (modern)
