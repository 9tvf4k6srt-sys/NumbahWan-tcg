## Global Objects API

### NW_WALLET
```javascript
NW_WALLET.init()                    // Initialize wallet
NW_WALLET.getBalance('wood')        // Get currency (wood/gold/gems)
NW_WALLET.spend('wood', 100)        // Spend currency
NW_WALLET.earn('wood', 50)          // Earn currency
NW_WALLET.isGM                      // Check if GM mode active
NW_WALLET.activateGM('numbahwan-gm-2026')  // Enable GM mode
NW_WALLET.getCollection()           // User's owned cards
NW_WALLET.getGuestId()              // User identifier
```

### NW_CARDS
```javascript
await NW_CARDS.init()               // Load card data
NW_CARDS.getAll()                   // All cards array
NW_CARDS.getBySeason(1)             // Cards from specific season
NW_CARDS.getByRarity('mythic')      // Filter by rarity
NW_CARDS.getById('s1-001')          // Single card lookup
```

### NW_JUICE (Game Feel)
```javascript
NW_JUICE.haptic.light()             // Light vibration
NW_JUICE.haptic.heavy()             // Heavy vibration
NW_JUICE.screen.shake('medium')     // Screen shake
NW_JUICE.sound.play('flip')         // Play sound effect
```

### NW_NAV
```javascript
NW_NAV.init()                       // Initialize nav (auto-called)
NW_NAV.open()                       // Open menu
NW_NAV.close()                      // Close menu
NW_NAV.setLang('zh')                // Change language
```

## GM Mode

### Activation Methods
1. **Console**: `NW_WALLET.activateGM("numbahwan-gm-2026")`
2. **URL param**: Add `?gm=true` to any page
3. **Whitelist**: Add Guest ID to `GM_WHITELIST` in nw-wallet.js

### GM Features
- Infinite currency (pulls don't cost logs)
- Shows `👑 GM` badge
- Access to admin APIs
- Persistent via localStorage

## Common Issues & Fixes

### "X is not defined" errors
- Check script load order (essentials → wallet → cards → page-specific)
- Ensure scripts are in `<head>` or before usage

### GM mode not working
- Call `NW_WALLET.init()` first
- Check localStorage: `localStorage.getItem('nw_gm_mode')`

### Cards not loading
- Verify JSON file exists: `/static/data/cards-v2.json`
- Check network tab for 404s
- Call `await NW_CARDS.init()` before `getAll()`

### Pack animation broken
- Check for duplicate overlay hide calls
- Verify `packOpeningResolve` promise flow
