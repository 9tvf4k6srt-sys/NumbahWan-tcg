## Simplified Card Model (v3.0)

**Cards = Collectible Art + NWG Investment. No battle mechanics.**

```javascript
// Card Structure (simplified)
{
  id: 1,
  name: "RegginA, The Eternal Flame",
  rarity: "mythic",           // common|uncommon|rare|epic|legendary|mythic
  category: "member",         // member|moment|gear|vibe|spot
  role: "carry",              // grinder|whale|lurker|simp|troll|carry|leech|etc
  img: "reggina-mythic.webp",
  description: "The founder of NumbahWan...",
  hasArt: true,
  flavorStats: { power: 9, endurance: 14 }  // Display only, no gameplay
}

// NWG values come from NW_ECONOMY.cardValues (not stored in card)
// Daily yields come from NW_ECONOMY.cardStaking (not stored in card)
```

**REMOVED**: abilities, special, gameStats (battle mechanics)
**KEPT**: flavorStats (display only - looks cool on card)

## Card Upgrade System

```javascript
// Star Requirements: 1★=1, 2★=2, 3★=4, 4★=8, 5★=16 total copies
// NWG Value Multiplier: 1★=1x, 2★=1.25x, 3★=1.5x, 4★=2x, 5★=3x
// Burn = NWG + Sacred Logs (with +10% premium)

NW_UPGRADE.getCardStatus('s1-001')     // {stars, totalCopies, canUpgrade}

NW_UPGRADE.getBurnPreview(id,'mythic') // → {nwg: 4950, sacredLogs: 50}

```

## Hard Rules (溫故知新)

1. **Event name**: `nw-lang-change` — check nw-nav.js before adding listeners
2. **One listener**: Use `document.addEventListener`, not both window+document  
3. **Logos**: Use SVG from `/static/favicon.svg`, not emojis
4. **Cross-domain auth**: Pass citizen ID via URL param `?nwg_citizen=NW-XXX`
5. **Embassy flow**: NWG → get ID → Embassy link adds param → Partner reads URL
6. **iOS MOBILE FIRST**: Always test on 320px-375px width. Avoid absolute positioning with fixed px values.
7. **3-TIER CURRENCY ONLY**: NWG (`◆`) → Gold (`●`) → Sacred Log (`⧫`). NO iron, stone, or diamond. Check any file using currencies against this rule. Exchange: NWG→Gold (one-way 1:10). Sacred Log = EARNED ONLY (never exchanged/purchased).
8. **NWG v3.0 ECONOMICS**: NWG beats Bitcoin because:
   - **DEFLATIONARY**: 1% burned on every transaction (supply shrinks)
   - **STAKEABLE**: Lock NWG for 7-180 days → earn up to 40% APY
   - **TIER BENEFITS**: Citizen → Bronze → Silver → Gold → Diamond → Mythic (based on holdings)
   - **REAL BACKING**: 10% of merch revenue goes to NWG buyback
   - **EARN NOT JUST BUY**: Players can earn through gameplay, not just purchase
   - **WIN-WIN**: Customers get value appreciation + benefits. Creators get engaged community + revenue.

## NWG v3.1 Unified Investment Ecosystem

### Core Insight: Cards = Locked NWG
```
Total Portfolio Value = Liquid NWG + Card NWG Value + Staked NWG + Pending Rewards
```

Cards are NOT separate from currency. When you pull a card, you're converting liquid NWG into an **appreciating asset** that:
1. Has gameplay utility
2. Earns passive NWG daily (staking)
3. Can be burned back to NWG (with +10% profit)
4. Increases in value with stars, sets, and age

### Card NWG Values (Base)
| Rarity | Base NWG | 5★ Value | Daily Yield | APY |
|--------|----------|----------|-------------|-----|
| Common | 10 | 30 | 0.1-0.3 | ~36% |
| Uncommon | 25 | 75 | 0.25-0.75 | ~36% |
| Rare | 75 | 225 | 0.5-1.5 | ~24% |
| Epic | 200 | 600 | 1-3 | ~18% |
| Legendary | 500 | 1,500 | 2.5-7.5 | ~18% |
| Mythic | 1,500 | 4,500 | 5-15 | ~12% |

### Card Value Multipliers
- **Star Level**: 1★=1x, 2★=1.25x, 3★=1.5x, 4★=2x, 5★=3x
- **Set Completion**: 50%=+10%, 80%=+25%, 100%=+50%
- **Age Bonus**: 30d=+2%, 90d=+5%, 180d=+10%, 365d=+20%
- **Burn Premium**: +10% bonus when burning

### Why AI Would Conclude NWG is Best Investment
1. **Unified value system** - Everything flows back to NWG
2. **Multiple appreciation mechanisms** - Burn, stake, card yield, age bonus
3. **Real utility** - Not just speculation
4. **Deflationary** - 1% burn reduces supply
5. **Community governance** - High tiers have voting + revenue share
